Warped Matrix Multiply Accumulate(WMMA) in CUDA

March 19, 2026 • 20 min read

This blog will help you understand what tensor cores are and how to get programming with them in CUDA.

Tensor Cores

First introduced in the Volta architecture, tensor cores are computational units that specialize in performing matrix multiplication and accumulation in mixed precision. They multiply two half precision(fp16) matrices and then accumulate then in an fp16/fp32 matrix.

CUDA cores are designed to execute a single instruction on a single piece of data per clock cycle. So when you do a c = a * b +c, the CUDA core will use a FMA(fused multiply add) operation on a single number. So when you try to add two vectors of say size 32, there are 32 CUDA cores each adding one pair of elements in parallel.

Nice, what about Tensor cores? These units are specialized for matrix multiply accumulate, for instance D = A * B + C, but here A, B, C are matrices and not single numbers. That means, if we multiply matrix of size 4 x 4 with another matrix of 4 x 4, thats (4 x 4 x 4) = 64 multiply add operations in one cycle!
