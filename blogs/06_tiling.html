Matrix Multiplication

Well, just as a reminder, at its core, matrix mulplication computes each output element as a dot product:

$$ c_{ij} = \sum_{k=1}^{n} a_{ik} \cdot b_{kj} $$

Naive Matrix Multiplication

In this implementation, we simply load the rows of matrix A and the columns of matrix B and compute a dot product over them.

Lets take a look at this visually with a small matrix


__global__ void matmul(float* A, float* B, float* C, const int M, const int K, const int N){
    int t_row = blockIdx.y * blockDim.y + threadIdx.y;
    int t_col = blockIdx.x * blockDim.x + threadIdx.x;

    if(t_row < M && t_col < N){
        float tmp = 0.0f;
        for(int i = 0; i < K; i++){
            tmp += A[t_row * K + i] * B[i * N + t_col];
        }

        C[t_row * N + t_col] = tmp;
    }
}

Whats going on in the kernel?

int t_row = blockIdx.y * blockDim.y + threadIdx.y; int t_col = blockIdx.x * blockDim.x + threadIdx.x; these two lines first index into the rows and cols of the matrices appropriately
